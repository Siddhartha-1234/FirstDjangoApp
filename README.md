
# Textutils Django app

It is a Django app for Basic text work
## Installation 

Make sure that [Django]("djangoproject.com") and [Python]("python.org") is installed

* Install Django with
```bash 
  pip install django
```
    
# Run
To run this project

```bash
  python manage.py runserver
```
